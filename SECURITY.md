# Security Policy

## Reporting a vulnerability

Please report vulnerabilities privately by opening a GitHub Security Advisory in this repository.

Include:

1. A concise description of the issue.
2. Affected versions/commit range.
3. Reproduction steps or proof of concept.
4. Impact assessment.

## Scope notes

This project can execute TFVC and PowerShell commands. Keep machine-specific data, credentials, and private infrastructure details out of committed files and test fixtures.
